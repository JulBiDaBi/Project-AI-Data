# Expérimentations en Détection et Suivi d'Objets

Ce dossier regroupe plusieurs projets explorant différentes architectures et méthodes de détection d'objets.

## Projets Inclus

1.  **Détection d'Animaux** (`Project_AnimalsDetection.ipynb`) :
    Utilisation de modèles pré-entraînés pour identifier diverses espèces animales.
2.  **Détection de Visages Humains** (`Project_HumanFaceDetection with YOLO.ipynb`) :
    Mise en œuvre de **YOLO** pour la localisation précise de visages dans des images et vidéos.
3.  **Suivi d'Objets Multiples (MOT)** (`Project_MultiObjectTracking with Ultralytics.ipynb`) :
    Utilisation de la bibliothèque Ultralytics pour suivre des objets à travers les images d'une séquence vidéo.
4.  **Détection Avancée avec Grounding DINO** (`Project MBY_with_Grounding_DINO.py`) :
    Exploration de la détection basée sur des descriptions textuelles (Open-Vocabulary Object Detection).

## Technologies Utilisées

- YOLO (v8, v11)
- Grounding DINO
- OpenCV
- PyTorch
- Ultralytics
