# Détection de Gestes de la Main en Temps Réel

Ce projet utilise la vision par ordinateur pour détecter et reconnaître des gestes de la main à partir d'un flux vidéo (webcam).

## Fonctionnalités

- **Accès Webcam** : Intégration du flux vidéo en temps réel.
- **Points de Repère** : Identification des points clés de la main (articulations, poignet) grâce à **MediaPipe Hands**.
- **Reconnaissance de Gestes** : Analyse de la position des doigts pour identifier des signes spécifiques (exemple : signe "OK").

## Technologies

- Python
- OpenCV
- MediaPipe
- JavaScript/HTML (pour l'affichage dans le notebook)
