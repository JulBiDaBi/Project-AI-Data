# Portfolio de Projets IA, Data & Computer Vision

Ce dépôt regroupe plusieurs projets réalisés dans les domaines de l'Intelligence Artificielle, de la Science des Données et de la Vision par Ordinateur.

## Structure du Projet

Le dépôt est organisé comme suit :

### 1. [Medical ChatBot](./1.%20Medical%20ChatBot/)
Un chatbot médical intelligent utilisant une architecture **RAG (Retrieval-Augmented Generation)**.
- **Technologies** : Python, Flask, Streamlit, LangChain, Pinecone, HuggingFace (FLAN-T5).
- **Fonctionnalités** : Extraction de texte depuis PDF, recherche sémantique, réponses basées sur des documents médicaux.

### 2. [Portfolio](./2.%20Portfolio/)
Une application web moderne présentant l'ensemble de mes travaux.
- **Technologies** : React, Vite, Tailwind CSS.
- **Fonctionnalités** : Design réactif, mode sombre, présentation détaillée des projets.

### 3. [Computer Vision](./3.%20Computer%20Vision/)
Une collection de projets et d'expérimentations en vision par ordinateur.
- **0. Module Introduction CV** : Construction d'un CNN à partir de zéro.
- **1. Classification** : Classification de tumeurs cérébrales par IRM.
- **2. Hands Gestures Detection** : Détection de gestes de la main en temps réel avec MediaPipe.
- **3. Object Detection** : Détection d'animaux, de visages et suivi d'objets multiples (YOLO, Grounding DINO).
- **4. Real-Time MultiVision** : Système multitâche de reconnaissance d'émotions et d'objets.

### 4. [Power BI Projects](./4.Power%20BI%20Projects/)
Analyses de données et tableaux de bord interactifs.
- *Dossier en cours de remplissage.*

## Comment utiliser ce dépôt

Chaque sous-répertoire contient ses propres instructions d'installation et d'utilisation. En général :
1. Clonez le dépôt.
2. Accédez au dossier du projet souhaité.
3. Suivez le `README.md` spécifique au projet.
